Mis Habilidades

Como administrador de sistemas y redes con especialidad en ciberseguridad, he desarrollado competencias en áreas críticas para la gestión de infraestructuras tecnológicas

Administración de Redes

Gestión completa de infraestructuras de red, configuración de routers, switches, firewalls, VLANs, y resolución de problemas de conectividad. Experiencia en redes LAN/WAN, protocolos TCP/IP, DNS, DHCP, y monitoreo proactivo del rendimiento de la red para garantizar la disponibilidad del servicio.

Herramientas:

  • Wireshark para análisis de tráfico
  • Nmap para escaneo de puertos
  • Cisco Packet Tracer para simulación
  • Pfsense/OPNsense como firewalls

Administración de Sistemas

Administración de servidores Windows y Linux, gestión de usuarios, permisos, backups automatizados, actualizaciones de seguridad, y monitoreo de rendimiento. Experiencia con entornos virtualizados (VMware, Hyper-V), contenedores Docker, y gestión de servicios críticos como Active Directory, DNS, y servicios de archivo.

Tecnologías:

  • Windows Server 2016/2019/2022
  • Ubuntu Server, CentOS, Debian
  • VMware ESXi, Hyper-V
  • Active Directory, DNS, DHCP

Ciberseguridad

Implementación de políticas de seguridad, auditorías de vulnerabilidades, gestión de firewalls, detección de intrusiones (IDS/IPS), respuesta a incidentes de seguridad, y cumplimiento normativo. Especializado en protección de datos sensibles, hardening de sistemas, y creación de entornos seguros para aplicaciones críticas de negocio.

Metodologías:

  • Análisis de vulnerabilidades con OpenVAS
  • Hardening de sistemas (CIS benchmarks)
  • Gestión de parches de seguridad
  • Respuesta a incidentes (IR)
